Prestasi Mahasiswa

Tentang Program Studi

Program Studi Informatika mempelajari berbagai bidang komputasi modern seperti algoritma, rekayasa perangkat lunak, kecerdasan buatan, pengembangan web dan mobile serta keamanan sistem informasi. Mahasiswa tidak hanya belajar teori tetapi juga melakukan praktik melalui proyek dan penelitian teknologi.

Kegiatan Mahasiswa

Mahasiswa aktif mengikuti seminar teknologi, workshop pemrograman, kompetisi hackathon serta kegiatan penelitian bersama dosen. Kegiatan ini bertujuan meningkatkan kemampuan praktis mahasiswa

Timeline Kurikulum Informatika

Tahun Pertama

Mahasiswa mempelajari dasar ilmu komputer seperti algoritma, logika pemrograman, matematika komputasi, dan pengenalan sistem komputer. Materi ini menjadi fondasi utama sebelum mahasiswa mempelajari teknologi yang lebih kompleks.

Tahun Kedua

Mahasiswa mulai mempelajari teknologi pengembangan sistem seperti basis data, struktur data, pemrograman berorientasi objek, serta pengembangan aplikasi web modern.

Tahun Ketiga

Mahasiswa mendalami teknologi lanjutan seperti artificial intelligence, data science, jaringan komputer, serta keamanan sistem informasi.

Tahun Keempat

Mahasiswa mengikuti program magang industri serta melakukan penelitian skripsi untuk mengaplikasikan pengetahuan yang telah dipelajari.

Kelompok Keahlian Informatika

TMG

Teknologi Multimedia dan Game

Kelompok keahlian ini berfokus pada pengembangan teknologi multimedia interaktif serta industri game modern. Mahasiswa mempelajari desain grafis digital, animasi komputer, pemrograman game, serta teknologi virtual dan augmented reality. Bidang ini mempersiapkan lulusan untuk bekerja dalam industri kreatif digital seperti pengembangan game, multimedia interaktif, serta produksi konten digital berbasis teknologi.

ISI

Informatika dan Sistem Inteligen

Kelompok keahlian ini mempelajari teknologi kecerdasan buatan (Artificial Intelligence) dan sistem cerdas yang mampu membantu pengambilan keputusan berbasis data. Mahasiswa akan mempelajari machine learning, data mining, computer vision, serta sistem rekomendasi yang banyak digunakan dalam industri teknologi modern.

JKF

Jaringan Komputer dan Digital Forensik

Kelompok keahlian ini mempelajari infrastruktur jaringan komputer, keamanan sistem informasi, serta teknik investigasi digital. Mahasiswa akan mempelajari keamanan jaringan, ethical hacking, digital forensic investigation, serta sistem keamanan data yang digunakan oleh perusahaan maupun instansi pemerintahan.

Prospek Karir Lulusan Informatika

Software Engineer

Mengembangkan aplikasi perangkat lunak skala besar seperti sistem perusahaan, aplikasi web, maupun platform digital modern.

Web Developer

Membangun dan mengembangkan website modern serta sistem berbasis web menggunakan teknologi frontend dan backend.

Data Scientist

Menganalisis data dalam jumlah besar untuk menemukan pola dan menghasilkan insight yang membantu pengambilan keputusan.

Mobile Developer

Mengembangkan aplikasi mobile berbasis Android maupun iOS yang digunakan oleh jutaan pengguna.

Cyber Security Specialist

Melindungi sistem informasi dari ancaman keamanan digital serta melakukan analisis kerentanan sistem.
